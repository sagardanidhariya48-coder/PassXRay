## 💡 Why This Project?

Weak passwords are one of the most common causes of security breaches.
This project helps users understand password strength and encourages better security practices.

---

## 🧠 How It Works

The password is analyzed based on:

* Length (minimum 8 characters)
* Uppercase letters
* Lowercase letters
* Numbers
* Special characters

Each condition increases the strength score.
